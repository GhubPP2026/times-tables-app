# 🌟 Times Tables Adventure

A fun, production-ready multiplication practice app for kids (1–10 times tables).

## ✨ Features
- 👥 Multi-player with persistent progress saved in browser
- 📚 3 learning techniques per table: Skip Count, Dot Arrays, Memory Tricks
- 🎯 20-question quiz with big number pad (perfect for 7-year-olds)
- ⚡ Speed tracking & personal best records
- 🎉 Confetti, sounds, and streak celebrations
- 📊 Full progress dashboard per player
- 📱 Works on any device — phone, tablet, computer
